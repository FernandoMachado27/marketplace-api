# 🛒 E-commerce Platform (Study Project)

Projeto de estudo desenvolvido com o objetivo de simular uma plataforma de e-commerce real, inspirado em grandes players do mercado como o Mercado Livre.

A proposta é evoluir o sistema gradualmente, aplicando boas práticas de desenvolvimento backend, arquitetura e design de APIs, com foco em aprendizado contínuo.

---

## 🚀 Objetivo

Construir uma aplicação backend robusta, próxima de um cenário real de mercado, explorando conceitos como:

- Arquitetura em camadas
- Boas práticas com APIs REST
- Modelagem de dados
- Tratamento de exceções
- Evolução para regras de negócio complexas

---

## 🧠 Tecnologias utilizadas

- Java 17+
- Spring Boot
- Spring Web
- Spring Data JPA
- Banco de Dados Relacional (ex: PostgreSQL / H2)
- Lombok
- Maven

---

## 🏗️ Arquitetura

O projeto segue o padrão de arquitetura em camadas:

- **Controller**: Responsável por expor os endpoints da API
- **Service**: Contém as regras de negócio
- **Repository**: Comunicação com o banco de dados
- **DTOs**: Camada de transferência de dados para evitar acoplamento

---

## ✅ Funcionalidades implementadas

- ✔️ Criação de usuários (API REST)
- ✔️ Integração com banco de dados
- ✔️ Persistência de dados
- ✔️ Uso de DTOs para entrada e saída
- ✔️ Tratamento de exceções centralizado
- ✔️ Estrutura organizada e escalável

---

## 🔄 Próximos passos

## 💡 Motivação

Este projeto foi criado com o objetivo de evoluir como desenvolvedor backend, saindo do nível básico e aplicando conceitos que são utilizados no mercado.

Mais do que aprender teoria, a ideia é construir algo real, enfrentando problemas reais.
